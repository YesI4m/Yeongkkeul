# 영끌
## 영화가 끌리다

영화 애호가들을 위한 영화 리뷰 커뮤니티



## 개발 팀
``
총 3명
``



## 기술 스택

- FrontEnd - HTML5/CSS3, JavaScript, BootStrap
- BackEnd - JSP
- DB - MySQL


## 주요 기능

| 기능 | 설명 |
| ------ | ------ |
| 회원 | 회원가입, 로그인, 프로필 관리, 이메일 인증 |
| 영화 관리 | 영화 조회, 영화 추가/수정/삭제, 좋아요 |
| 댓글 | 영화 리뷰, 댓글 삭제, 좋아요, 최근 댓글, 좋아요한 댓글 모아보기|
| 검색 | 영화 검색, 사용자 검색, 최근 검색어 기록, 인기 검색어 |

## 개발 환경

> JAVA 1.8
> 
> IDE : Oracle
