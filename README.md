# 영끌
## 영화가 끌리다

영화 애호가들을 위한 영화 리뷰 커뮤니티



## 개발 팀
``
총 3명
``



## 기술 스택

- FrontEnd - React native(Expo)
- BackEnd - Spring Boot 3, FastAPI
- DB - PostgreSQL(Supabase)
- Gpt 4.0 Turbo API
백엔드 API 작업 중 POSTMAN 사용

## 주요 기능

| 기능 | 설명 |
| ------ | ------ |
| 소셜 로그인 | 카카오, 네이버 소셜 로그인 지원 |
| 만보기 | 사용자 운동량 확인 |
| 식단, 운동량 추천 | GPT 4.0 API를 사용해 사용자에 맞는 식단과 운동량 추천 및 지난 추천기록 확인|
| 달력,일기 | 하루 운동량을 달성했는지 확인, 일기 작성 |
| 펫 | 운동량 달성 여부에 따라 먹이 획득 가능, 먹이에 따른 펫 상태 변화 |
| 서버 이중화 | Spring Boot와 FastAPI 두개의 서버로 한쪽 서버가 꺼져도 동일한 기능을 유지 가능  |
